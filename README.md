# Simple Calculator

This is a simple command-line calculator application written in Python. It can evaluate mathematical expressions and supports variables.

## Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/snyk/simpleCalculator.git
    cd simpleCalculator
    ```

2.  **Create a virtual environment:**
    ```bash
    python3 -m venv venv
    source venv/bin/activate
    ```

3.  **Install the dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

## Usage

To start the calculator, run the following command:
```bash
python calculator.py
```

You can then enter mathematical expressions at the prompt:
```
> 2 + 2
4
> 3 * (4 + 5)
27
> pi
3.141592653589793
```

To exit the calculator, type `exit`.
